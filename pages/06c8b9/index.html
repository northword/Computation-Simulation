# 可能需要安装的软件

一些WINDOWS上可能会用到的软件：

Software Name Description Comment
XShell / VS Code SSH Romate SSH远程连接工具 官网找教育版 (opens new window)
XFTP FTP/SFTP文件传输工具 同上
Xming / XManager X Window远程连接工具 Xming免费 下载 (opens new window)
Materials Studio 可视化建模；MS本身可以进行计算，但不常用
VESTA 结构模型，体积数据（如电子/核密度）和晶体形态的3D可视化程序 官网 (opens new window) / 下载 (opens new window)
Python
ASE 原子模拟环境 官网 (opens new window) / 安装见windows-ase-install.md
一个好的文本编辑器 eg. NotePad++（NOT Recommend because its author is anti-Chinese） 呱~

这是一些在Linux上可能用得到的软件

Software Name Description Comment
VASP Vienna Ab-initio Simulation Package, 电子结构计算和量子力学-分子动力学模拟软件包 官网 (opens new window) / 集群管理员已编译
VTST Vasp TST(transition state theory) Tools 官网 (opens new window) / 分Code和Script两部分，Code集群管理员已编译
VTST-Script 安装见 linux-install-VTSTScript.md
VASPKIT VASP预-后数据处理脚本 官网 (opens new window) / 安装见 install-vaspkit.md
Python 学校配置了python27
Irzsz 一个UNIX通信软件包，提供XMODEM，YMODEM和ZMODEM文件传输协议。 官网 (opens new window) / 安装见 linux-lrzsz-install.md
gunplot 命令行的交互式绘图工具 官网 (opens new window) / 学校已安装 / 安装见 linux-gunplot-install.md
ase 安装见 windows-ase-install.md
